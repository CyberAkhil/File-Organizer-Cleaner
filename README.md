# File Organizer & Cleaner

A Python automation script that organizes files in a directory
based on file type such as images, documents, videos, and music.

## Features
- Automatically sorts files into folders
- Uses OS and shutil modules
- Handles unknown file types

## Technologies Used
- Python
- OS Module
- File Handling

## How to Run
1. Place the script in any directory
2. Run `python organizer.py`
3. Enter the folder path to organize
